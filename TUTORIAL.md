## 三步，五分钟

### 第一步：装 App
[GitHub Actions](https://github.com/xiaoxuan0820-ctrl/AegisForge/actions) → 下载最新 APK → 装到旧手机

> Android 8.0+ 即可，不需要 Root

### 第二步：开权限
设置 → 辅助功能 → 已安装的服务 → CiCi → 开启

### 第三步：配 Key
电脑浏览器打开 `http://手机IP:9527` → 填 DeepSeek API Key → 保存

> DeepSeek API 注册就送额度，够你用几个月

然后打开飞书/钉钉，给 Bot 发一句话。手机亮了。它开始动了。
